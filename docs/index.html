LINE Bot 資料流

LINE Bot 不是只有 LINE。真正重點是資料怎麼流動。

1. 使用者輸入

使用者傳文字、照片、發票號碼或報修內容。

2. Webhook

LINE 把事件送到你設定的 Webhook URL。

3. 後端處理

Apps Script、伺服器或自動化工具接收資料並判斷。

4. 寫入資料

把發票、報修或巡檢資料寫入 Google Sheet。

5. 回覆使用者

Bot 回覆：「已收到」、「已建立案件」、「查詢成功」。

6. 後續通知

中獎、結案、逾期或異常時自動提醒。
