# 資工工具學習地圖

這個專案用來練習常見的資工工具與工作流程。

## 第一階段：GitHub 基礎

目標：知道 GitHub 是什麼，會建立專案、修改檔案、留下紀錄。

我要學會：

- Repository 是什麼
- README 是什麼
- Commit 是什麼
- 如何建立資料夾
- 如何建立新檔案
- 如何修改檔案

## 第二階段：專案整理

目標：把資料整理成一個清楚的專案。

我要學會：

- docs 放說明文件
- src 放程式或主要內容
- outputs 放成果
- data 放資料
- README 寫專案首頁

## 第三階段：網頁部署

目標：把一個簡單網頁放到網路上。

我要學會：

- HTML 是什麼
- GitHub Pages 是什麼
- 如何讓別人看到我的網頁

## 第四階段：LINE Bot

目標：理解 LINE Bot 的基本結構。

我要學會：

- LINE Bot 是什麼
- 使用者傳訊息後發生什麼事
- Webhook 是什麼
- 後端伺服器是什麼
- API 是什麼
