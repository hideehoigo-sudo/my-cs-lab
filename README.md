CS Practice

這是一個個人資工工具練習專案。

本專案的目標不是一開始就大量寫程式，而是先理解資工常用工具的用途與串接方式，例如 GitHub、GitHub Pages、LINE Bot、Webhook、Google Sheet、API、自動化工具等。

專案目標

我希望透過這個專案學會：

使用 GitHub 管理專案
使用 Issue 拆解任務
建立清楚的資料夾結構
部署一個簡單網頁
理解 LINE Bot 的基本架構
理解 Webhook 是什麼
使用 Google Sheet 作為簡易資料庫
設計報修、發票、巡檢等實用工具流程
